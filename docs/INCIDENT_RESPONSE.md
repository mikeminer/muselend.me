# Incident response

Severity 0 covers active asset loss or governance compromise; immediately pause new risk,
notify the Safe signers/security contacts, preserve evidence and publish a factual status
notice. Never rotate or disclose user keys. Recovery must prioritize permissionless exits
and protected reserves. Postmortems include timeline, impact, root cause, remediation,
tests and monitoring changes. Contact roster and disclosure SLAs are launch gates.
